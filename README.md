CREDITI CARD EXPENDITURE FORECASTING

Objective: - Enhance marketing efforts to boost average credit card spending effectively.

Business Problem: -
	The primary objective is to boost credit card spending among potential clients. This insight enables banks to tailor services and develop effective strategic marketing plans.
Data availability: -
	Customer Demographics data ( Customer details like age , gender , employment experience etc)
	Customer Behaviour data ( Customer consumption expenditure data of 3months )
	Customer Consumption ( Customer id and consumption data)

Tools used  :-  python (Pandas , NumPy, MatPlot, Seaborn ,Sklearn )
Techniques  :-
	Exploratory Data Analysis & Data Transformation
	The objective of this phase was to clean the raw data and prepare the feature set for the machine learning pipeline. The process involved auditing 20,000 records across 28 features to ensure data integrity and statistical validity.
